เอนจินเดียวกัน หกเทมเพลต

  • dBA / dBC / dBZ แบบเรียลไทม์พร้อมการตอบสนอง Fast / Slow
  • สถิติ LAeq, L10, L90 และเกณฑ์แจ้งเตือน
  • ภาพหลักฐานเสียงที่มี dB สด GPS สถานที่ และบริบทการสอบเทียบ
  • เสียง วิดีโอ เครื่องหมายเหตุการณ์ด้วยตนเอง และบันทึกย่อ (ไม่บังคับ)
  • ส่งออก PDF, CSV และ JSON manifest — ทุกอย่างอยู่ในเครื่อง เว้นแต่คุณเลือกแชร์
SOUNDTEST.PRO ไม่ใช่เครื่องวัดระดับเสียงที่ได้รับการรับรอง สำหรับหลักฐานทางกฎหมาย กฎระเบียบ หรือข้อพิพาทอย่างเป็นทางการ กรุณาตรวจสอบด้วยอุปกรณ์มืออาชีพที่สอบเทียบแล้วและปฏิบัติตามขั้นตอนในท้องถิ่น
