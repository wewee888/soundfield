Suggested script

  1. Record before work begins.
  2. Record the loudest phase with photo or video.
  3. Record after work stops for comparison.
  4. Export PDF and save original media files.
SOUNDTEST.PRO is not a certified sound level meter. Project compliance, permits, or enforcement decisions should use calibrated instruments and qualified measurement procedures.
