Trust center · Local-first

Local-first privacy for sensitive noise records.

Noise evidence can include voices, people, private spaces, and precise locations. SOUNDTEST.PRO keeps the Stage 1 web beta on your device unless you explicitly export or save files.

Microphone

Live, on-device analysis

The microphone stream is processed in the browser for live dB analysis. Audio is saved only after you start audio or video recording.

Camera

Optional capture only

Used only for preview, acoustic evidence photos, or watermarked video. You control when media is generated.

Location

You choose the granularity

Used for GPS, map links, place labels, and report metadata. Reverse geocoding may send coordinates to the selected map provider.

Storage

localStorage + IndexedDB

Settings and metadata use localStorage. Media clips use IndexedDB when supported. Browser cleanup may remove local data.

Exports

Only when you say so

PDF, CSV, JSON manifest, images, audio, and video files are created only when you choose to export or save them.

Beta scope

No cloud in beta

Stage 1 has no account sync, cloud backup, team sharing, or server-side evidence retention.

User responsibilities

Stay safe and respectful while recording.

  • Follow local audio/video recording consent laws.
  • Avoid sharing faces, private spaces, exact residential locations, or sensitive details without permission.
  • Save important media promptly because browser storage can be cleared.
Future account or cloud versions must add account deletion, cloud deletion, retention controls, regional compliance, and encrypted storage policies before paid global launch.
