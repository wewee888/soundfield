Trust center

Local-first privacy for sensitive noise records.

Noise evidence can include voices, people, private spaces, and precise locations. SOUNDTEST.PRO keeps the Stage 1 web beta local unless users explicitly export or save files.

Microphone

Used for live sound analysis in the browser. Audio is saved only after the user starts audio or video recording.

Camera

Used only for preview, acoustic evidence photos, or watermarked video. The user controls when media is generated.

Location

Used for GPS, map links, place labels, and report metadata. Reverse geocoding may send coordinates to the selected map provider.

Local storage

Settings and record metadata use localStorage. Media clips use IndexedDB when supported. Browser cleanup may remove local data.

Exports

PDF, CSV, JSON manifest, images, audio, and video are created only when the user chooses to export or save them.

No cloud in beta

The Stage 1 web beta has no account sync, cloud backup, team sharing, or server-side evidence retention.

User responsibilities

  • Follow local audio/video recording consent laws.
  • Avoid sharing faces, private spaces, exact residential locations, or sensitive details without permission.
  • Save important media promptly because browser storage can be cleared.
Future account or cloud versions must add account deletion, cloud deletion, retention controls, regional compliance, and encrypted storage policies before paid global launch.
