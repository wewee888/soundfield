Distribution

Start with the web app. Install as PWA when your browser supports it.

SOUNDTEST.PRO is intentionally launching as a website first. The browser version validates demand before native app store billing, accounts, cloud backup, and background recording claims.

Web/PWA beta

Use directly from HTTPS. Add to Home Screen on iOS or install from Chrome/Edge when available.

Future desktop offline

Planned as a local-first one-time purchase for offline monitoring, local export, calibration profiles, and no ads.

Future iOS/Android

Native app store releases require platform billing, permission review material, native save flows, and real-device validation.

Browser requirements

  • Secure context: HTTPS, localhost, or 127.0.0.1.
  • Microphone permission for live sound analysis.
  • Camera permission only for evidence photos or video.
  • Location permission only for GPS, map links, and place labels.
  • IndexedDB for media persistence after refresh when supported.
PWA install support does not mean App Store readiness. Background recording, native file save, subscriptions, privacy labels, and review assets remain separate release work.
